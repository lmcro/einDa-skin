Een IP linken met apache zal de linked IP toevoegen aan de VirtualHost waar het huidige IP in staat. LANG_MULTI_IP_RELATION=Dit is onafhankelijk van het Multi-IP systeem, maar kan er wel naast gebruikt worden.
